# Projeto: Página de Carreira em Desenvolvimento Web

Este projeto é uma página web estática informativa, criada como parte do Projeto Integrador do curso de Análise e Desenvolvimento de Sistemas (ADS) da UNIG, com foco em Gestão de Carreiras.

A página apresenta um panorama técnico e estratégico sobre a carreira de Desenvolvimento Web, destinada a futuros profissionais de TI.

# Visão Geral

O site é uma "one-page" (página única) que aborda os seguintes tópicos:

* O que faz um profissional da área
* Tecnologias, linguagens e ferramentas (Front-end, Back-end, DB, DevOps)
* Certificações e qualificações valorizadas
* Média salarial e perspectivas de mercado no Brasil
* Empresas que contratam desenvolvedores
* Desafios e oportunidades da carreira
* Contribuição da área com o ODS 9 (Indústria, Inovação e Infraestrutura)
* Dicas para quem está começando
* Links de contato profissional (GitHub e LinkedIn)

# Tecnologias Utilizadas

Este projeto foi construído utilizando apenas tecnologias front-end fundamentais:

* **HTML5:** Para a estruturação semântica do conteúdo.
* **CSS3:** Para a estilização, utilizando:
    * Variáveis CSS para um tema de cores consistente.
    * Media Queries para garantir a responsividade em dispositivos móveis.
    * Google Fonts para a tipografia (`font-family: 'Inter'`).

# Como Visualizar

Por ser um site estático, não há necessidade de instalação ou build.

1.  Clone este repositório:
    ```sh
    git clone [https://github.com/seu-usuario/seu-repositorio.git](https://github.com/seu-usuario/seu-repositorio.git)
    ```
2.  Abra o arquivo `index.html` (ou o nome do seu arquivo HTML principal) em qualquer navegador web.
